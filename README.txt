====>Projeto de Controle de Ponto<=====

Instalação:

1: Instalar o Xampp 7.2.7-0-VC15

2: Instalar o node-v8.11.3

3: Instalar o Composer-Setup

Processo de Utilização do Sistema:

O Layout do sistema de controle de ponto foi desenvolvido para ter uma interface
amigável e simples. O sistema possui um relógio, um campo de texto e doze botões: 
dez botões com valores de 0 à 9 e dois botões: um de limpar o que foi pressionado 
e outro para confirmar o valor inserido. 

Todos esses componentes estão presentes à esquerda da página. O lado direito seria 
a área do Administrador, onde a partir de uma senha padrão (0000) e após apertar o
botão "Confirmar", iriai abrir duas opções para o administrador: a primeira opção 
seria para cadastrar um novo funcionário e a segunda opção seria para exibir o rela-
tório do funcionário após uma pesquisa rápida.

Quando um funcionário inserir sua senha, uma mensagem apareceria no campo texto 
exibindo alguma informação a depender do horário do sistema.

Como não foi desenvolvido o banco de dados, o desenvolvimento do sistema não foi con-
cluído no prazo determinado, devido ao curto prazo de tempo para o estudo da nova ferra-
menta.

A tecnologia utilizada foi o Laravel v5.6 com Php -v 7.2.7. A escolha dessa ferramenta 
para o projeto foi para aprender e aperfeiçoar a experiência como desenvolvedor web.
